# solid-ts and tailwind

A base template for Solid projects built with TypeScript and Tailwind. It's includes a base linting and formatting setup.

## Dependencies

- [ESLint](https://eslint.org/)
- [Prettier](https://prettier.io/)
- [Tailwind CSS](https://tailwindcss.com/)
- [TypeScript](https://typescriptlang.org/)
- [Vite](https://vitejs.dev/)
